# ⏱️ Stopwatch Web Application

A simple and interactive **Stopwatch Web App** built using **HTML, CSS, and JavaScript** as part of my **Web Development Internship at Prodigy Infotech**.

## ✨ Features

- ▶️ Start the stopwatch  
- ⏸️ Pause functionality  
- 🔄 Reset timer  
- 🏁 Record lap times  
- ⏱️ Accurate time tracking  
- 📱 Responsive design  

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript** – Functionality
